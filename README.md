## Descrizione del Progetto

Questo repository raccoglie gli elaborati richiesti per il progetto finale del corso. Il lavoro si focalizza sull'analisi e la gestione del **Global Power Plant Database** (WRI), affrontando l'intero ciclo di vita del dato: dalla modellazione concettuale alla costruzione di un database relazionale, fino all'analisi predittiva avanzata.

Il progetto è strutturato per soddisfare i seguenti quattro requisiti fondamentali:

### 1. Exploratory Data Analysis (EDA)
L'attività prevede l'esplorazione approfondita del dataset fornito. Nello specifico:
* Descrizione della distribuzione dei dati.
* Calcolo delle metriche statistiche appropriate.
* Generazione di grafici associati.

### 2. Data Warehousing & Storage
Fase dedicata alla strutturazione e memorizzazione efficiente delle informazioni:
* Progettazione di un formato e di una struttura dati idonea.
* Implementazione fisica all'interno di un Database Relazionale (PostgreSQL) al fine di garantire l'integrità dei dati e l'efficienza nelle interrogazioni.

### 3. Visualizzazione Geospaziale
Sviluppo di strumenti di mapping e dashboarding:
* Visualizzazione su mappa della dislocazione geografica delle centrali elettriche a livello globale.
* Creazione di statistiche visuali aggregate per Paese e per tipologia di combustibile.

### 4. Forecasting & Machine Learning
Sviluppo di modelli predittivi per stimare la generazione futura di energia, con un orizzonte temporale di 3 anni. Le previsioni sono effettuate a tre livelli di aggregazione:
* Per singola centrale.
* Per tutte le centrali di un dato Paese (aggregato).
* Per tutte le centrali di una data tipologia (aggregato).
